4 The element HTML

The head element

・headはhtmlの最初の要素で使われる。
・先頭要素のスタートタグが空であるか、先頭要素の内側の最初のものが要素であるならば、省略することができる。
・また、先頭要素の後にアスキー空白文字か、コメントがすぐに続くならば、先頭要素のエンドタグは省略することができる。

ex)

<!doctype html>
<html lang = en>
<head>
<title>A doctype with a short head</title>
</head>
<body>
・・・

The title element

・titleタグはメタデータである。
・headタグに含まれることが多い。

ex)

<title>Introduction to The Mating Rituals of Base</title>
・・・
<h1>Introduction</h1>
<p>This companion guide to the highly successful</p>
<cite>Introduction to Medieval Bee-Keeping</cite>book is...

The link element

・linkタグはメタデータである。
・linkタグはbody内に含まれることが多い。
・endタグはない

種類

href - ハイパーリンクのアドレス
crossorigin - 要素がクロス起動要請を処理する方法
rel - ハイパーリンクを含んでいる文書と転送先の資源の関係
media - 適用できるメディア
integrity - 完全性メタデータが[SRI]において完全性をチェック
hreflang - 結合された資源の言語
type - リファレンスを付けられた資源タイプのヒント
referrerpolicy - 要素で始められる獲得のためのリファラ方針
sizes - アイコン(rel="icon")のサイズ
imagesrcset - 異なる状況（例えば、高解像度展示、小型モニターなど）の使用イメージ(for rel="preload")
imagesizes - 異なるページ・レイアウトのためのイメージサイズ(for rel="preload")
as - プレロード要請のための潜在的目的地(for rel="preload" and rel="modulepreload")
blocking - 要素が潜在的に封鎖することができるかどうか
color - サイトのアイコンをカスタマイズする時に使う色(for rel="mask-icon")
disabled - リンクが使用不能かどうか

ex)

<link rel="author license" href="/about">
